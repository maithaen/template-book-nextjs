# ຄູ່ມືການປັບແຕ່ງ

ບົດນີ້ກວມເອົາວິທີປັບແຕ່ງ Book Template ໃຫ້ກົງກັບຕົວຕົນແບຣນ ແລະ ຄວາມຕ້ອງການສະເພາະຂອງທ່ານ.

## ແບຣນດິ້ງ

### ໂລໂກ້

ເພື່ອປ່ຽນໂລໂກ້, ອັບເດດ component `Header.tsx`:

```tsx
// components/layout/Header.tsx
<Link href="/" className="flex items-center gap-2">
  <YourLogoComponent />
  <span>ຊື່ແບຣນຂອງທ່ານ</span>
</Link>
```

### ສີ

ໂຄງການສີໄດ້ຖືກກຳນົດໂດຍໃຊ້ CSS variables ໃນ `globals.css`:

```css
:root {
  --primary: oklch(0.65 0.19 160);
  --primary-foreground: oklch(1 0 0);
  /* ເພີ່ມສີເພີ່ມເຕີມ... */
}
```

## ການພິມ

### ຟອນ

ແມ່ແບບໃຊ້ຟອນ Geist ເປັນຄ່າເລີ່ມຕົ້ນ. ເພື່ອປ່ຽນຟອນ:

1. ນຳເຂົ້າຟອນຂອງທ່ານໃນ `app/layout.tsx`
2. ອັບເດດ CSS variables

```tsx
import { Inter } from "next/font/google";

const inter = Inter({
  variable: "--font-sans",
  subsets: ["latin"],
});
```

## ໂຄງສ້າງເນື້ອຫາ

### ການເພີ່ມບົດໃໝ່

1. ສ້າງໄຟລ໌ `.mdx` ໃໝ່ໃນໂຟນເດີ `content/`
2. ເພີ່ມ frontmatter ພ້ອມກັບຫົວຂໍ້, ລາຍລະອຽດ, ແລະ ລຳດັບ
3. ຂຽນເນື້ອຫາຂອງທ່ານເປັນ Markdown

### ລຳດັບບົດ

ຟິລ `order` ໃນ frontmatter ກຳນົດລຳດັບບົດ:

| ລຳດັບ | ບົດ           |
| ----- | ------------- |
| 1     | ການແນະນຳ      |
| 2     | ການປັບແຕ່ງ    |
| 3     | ຫົວຂໍ້ກ້າວໜ້າ |
| 4     | ການ Deploy    |

## ການແປ

### ການເພີ່ມພາສາໃໝ່

1. ສ້າງໂຟນເດີ locale ໃໝ່: `content/[locale]/`
2. ເພີ່ມໄຟລ໌ແປ: `lib/i18n/locales/[locale].json`
3. ອັບເດດ `LanguageContext.tsx` ເພື່ອລວມພາສາໃໝ່

```json
{
  "home": "ໜ້າຫຼັກ",
  "chapters": "ບົດ",
  "tableOfContents": "ສາລະບານ"
}
```

## ສະຫຼຸບ

ໃນບົດນີ້, ພວກເຮົາໄດ້ກວມເອົາ:

- ວິທີປັບແຕ່ງແບຣນດິ້ງ (ໂລໂກ້, ສີ)
- ການປ່ຽນແປງການພິມ ແລະ ຟອນ
- ໂຄງສ້າງເນື້ອຫາ ແລະ ການຈັດລະບຽບບົດ
- ການເພີ່ມພາສາໃໝ່

ຕໍ່ໄປ, ສຳຫຼວດທາງເລືອກການ deploy ໃນບົດ 3!
